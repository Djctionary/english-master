# v1.1-alpha — UI Refinement

Frontend-only visual refresh. No backend changes.

### 1. Landing Page — Multilingual Background
- "Sentence is all you need" in 20+ languages scrolling as background rows
- Multiple rows at different speeds and directions for depth
- Low opacity, decorative text creates cultural texture

### 2. Landing Page — Simplified Content
- Remove comparison table (verbose, takes too much space)
- Keep hero + 4-step "How it works" + CTA footer
- Tighter layout, stronger visual hierarchy

### 3. Auth Modal — Branded Header
- Add gradient header bar with app name
- Cleaner form spacing and visual treatment

### 4. UserNav — Avatar Redesign
- Replace "Hi, {username}" with circle avatar showing first initial
- Username displayed alongside, compact logout button
- Better integration into page header nav

### 5. Page Headers — Unified Layout
- Consistent header bar for Learn and Review pages
- Vertical divider separating user area from page nav
- Better alignment and spacing on all screen sizes

### 6. Global CSS
- New keyframe animations for scrolling background
- Avatar component styles
- Subtle entrance animations for page content
