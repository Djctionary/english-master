# v0.4 — UI/UX Design Overhaul

**Date:** 01/04/2026

## Goal

Unify the visual language across the entire app — establish a consistent design system (colors, typography, spacing, components) and fix existing UI bugs.

---

## Current State: Style Audit

### Problems Identified

1. **No unified color system** — 4+ shades of primary text (`#1F2937`, `#1D2D39`, `#1A2530`, `#111827`), 4+ shades of secondary text, inconsistent border colors. Brown/beige buttons (`#7b5136`) clash with blue action buttons (`#2563EB`).

2. **Typography chaos** — Section headings range from 11px to 16px with no hierarchy. Line heights vary (1.03, 1.4, 1.5, 1.6, 1.7, 2.0) without pattern. No consistent heading style system.

3. **Spacing inconsistency** — Section padding varies: `18px 20px`, `16px 20px`, `18px`. Border-radius mixes `999px`, `24px`, `8px`, `6px` without a defined scale.

4. **All styles are inline** — Components use hardcoded hex values in JSX. No CSS variables, no design tokens, no reusability.

5. **Weak responsive design** — `clamp()` used in some places but not others. Some padding values don't adjust on mobile.

6. **Bug: Long sentences overflow** — In SentenceLibrary, `white-space: nowrap` with no width constraint causes long text to push the delete button off-screen.

## Design Direction

### Art Style: "Calm Education"

A clean, warm, trustworthy aesthetic — inspired by the best of modern language learning apps. Not gamified like Duolingo, but approachable like Babbel. Think: a well-designed study tool, not a toy.

**Reference apps:**
- **Babbel** — clean, mature, structured layout, Roboto typography
- **Busuu** — minimal interface, clear hierarchy, timeline-style progress
- **Quizlet** — polished web-first, card-based, great use of whitespace
- **Linear** — (non-education) gold standard for clean SaaS UI: consistent spacing, subtle shadows, restrained color

**Inspiration galleries:**
- [Dribbble — Language Learning App](https://dribbble.com/tags/language-learning-app)
- [Behance — Language Learning App](https://www.behance.net/search/projects/language%20learning%20app)
- [Duolingo UI Kit (Figma)](https://www.figma.com/community/file/1377326303556981356/duolingo-free-ui-kit-by-marvilo)
- [Elingo UI Kit (Dribbble)](https://dribbble.com/shots/22968340-Elingo-Language-Learning-App-UI-Kit)

## Bug Fix

### Long sentence overflow in SentenceLibrary

**File:** `components/SentenceLibrary.tsx`
**Root cause:** Sentence text div uses `white-space: nowrap` inside a flex item without `min-width: 0`, so long text expands infinitely and pushes the delete button off-screen.
**Fix:** Add `min-width: 0` to the flex item button, add `overflow-wrap: break-word`, change `white-space` to `normal`.
